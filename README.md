# resume_screening

Project Overiew -
This project is a web-based application built using Streamlit that automatically screens and ranks resumes based on a given job description. It helps recruiters quickly identify the most suitable candidates by using Natural Language Processing (NLP) techniques.

How it works -
- Extracts text from uploaded PDF resumes
- Compares resumes with the job description
- Uses TF-IDF Vectorization and Cosine Similarity
- Ranks candidates based on relevance score

Technology Used -
- Python
- Streamlit
- Pandas
- Scikit-learn
- PyPDF2

Usage -
- Enter the job description
- Upload candidate resumes (PDF format)
- View ranked results instantly
